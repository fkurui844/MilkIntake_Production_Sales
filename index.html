📋 Excel File Format

Your Excel file should have:

  • Sheet name: "Milk Intake" or "Production" or "Sales"
  • Column A: Date (YYYY-MM-DD or standard date format)
  • Column B: Value (numbers)
  • Row 1: Headers (Date, Total in Ltrs, etc.)
  • Row 2+: Your data
